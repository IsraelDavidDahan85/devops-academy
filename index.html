How this site works

  • Each course is a folder under courses/. Click in to see its syllabus, homework labs, docs, and final project.
  • Every course is bilingual — English and Hebrew. The switcher in the top-right toggles language.
  • Slide decks live separately under slides/ and are served by the open-slide dev server — run pnpm dev, then navigate to /s/<slide-id>.
  • Inside any course folder, the slides/ entry is a symlink to its deck — same files, two access points.
  • Everything you see here is plain HTML + a shared assets/styles.css. No build step.

For maintainers

  • Program index for git/IDE — PROGRAM.md.
  • Authoring skill — .claude/skills/course-authoring/. Use phrases like "scaffold course 02", "write a homework lab for linux-fundamentals on FHS", "build the final for course 01".
