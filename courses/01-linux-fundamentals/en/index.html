Linux Fundamentals

The first hands-on course in the program. Students go from "I've never opened a terminal" to "I can stand up, operate, and troubleshoot a Linux box on my own" — covering the shell, shell power-ups, the system underneath, and the daily ops work of running a box.

Homework

Lab 01 — CLI Tour Guided shell tour: navigate, create, view, copy, clean up. ~45 min Lab 02 — Pipes & PATH Compose commands with pipes and globs; modify PATH. ~45 min Lab 03 — Boot & FHS Inspect the boot timeline, list systemd services, walk the FHS. ~60 min Lab 04 — Permissions & Users Create users, set up a shared group with SGID, walk SUID/sticky, run chmod scenarios. ~60 min Lab 05 — Storage & Services Build a loopback device, format and mount it, persist via /etc/fstab, write a .service + .timer. ~75 min

Final project

Make a Linux Box Yours Provision a fresh Linux VM, customize it end-to-end (PS1, aliases, PATH, ~/bin), add a non-root user, mount a volume, deploy a systemd service, document the boot, ship as a Git repo. 3–5 hr

Prerequisites

  • Course 00 — DevOps Intro (recommended, not strict).
  • A working Linux environment: a VM, WSL2, a cloud VM, or macOS terminal (with caveats — some paths differ).
