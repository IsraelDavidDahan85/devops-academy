# Course 01 — Linux Fundamentals

> The first hands-on course in the program. Students learn to live in a Linux shell: navigate the filesystem, manipulate files, read help pages, customize their environment, and understand what's actually happening under the hood — kernel, user space, boot, and the FHS.
